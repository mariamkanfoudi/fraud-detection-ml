# Détection de Fraudes Bancaires avec Python

## Description
Projet de Machine Learning pour détecter automatiquement les transactions
bancaires frauduleuses à partir de données réelles.  
Le dataset contient **284 807 transactions** dont seulement **492 fraudes (0.17%)** — 
un cas classique de déséquilibre de classes.

---

## 🛠️ Technologies utilisées
- Python 3.10
- Pandas / NumPy
- Scikit-learn
- XGBoost
- Matplotlib / Seaborn
- Google Colab
- SMOTE (imbalanced-learn)

---

## Dataset
[Credit Card Fraud Detection — Kaggle](https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud)

| Info | Valeur |
|------|--------|
| Transactions totales | 284 807 |
| Fraudes | 492 (0.17%) |
| Features | 30 (V1–V28 + Amount + Time) |
